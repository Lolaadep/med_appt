# grihf-frontend_capstone_starter_code

StayHealthy is a health-focused web application designed to help users track, manage, and improve their overall wellness. The project provides a simple and intuitive interface to support healthier lifestyle choices.

---

## 📌 Project Overview

StayHealthy helps users build and maintain healthy habits by providing tools for tracking wellness activities and staying consistent with personal health goals. It is built using modern web technologies for a smooth user experience.

---

## 🚀 Features

- Clean and responsive user interface  
- Health tracking dashboard  
- Easy navigation between pages  
- Mobile-friendly design  
- Fast and lightweight performance  

---

## 🛠️ Tech Stack

- React.js  
- Vite  
- JavaScript (ES6+)  
- HTML5  
- CSS3  

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

1. Clone the Repository
Open your terminal and run:

git clone https://github.com/Lolaadep/med_appt.git
2. Navigate into the Project Folder
cd med_appt
3. Install Dependencies
Make sure you have Node.js installed, then run:

npm install
4. Run the Frontend (React App)
npm run dev
This will start the React app.
You’ll see a local URL (e.g., http://localhost:5173). Open it in your browser.

5. Run the Backend (Server)
Open a new terminal and run:

cd server
node index.js
You should see:

Server is running on port localhost:8181
Connected to Mongo Successfully
6. Open the Application
Go to:

http://localhost:5173
Make sure the backend is running on port 8181.

🛠️ Notes
Ensure Node.js is installed

Ensure MongoDB is running (if required)
